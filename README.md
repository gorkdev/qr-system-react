# Akcan Grup QR Yönetim Paneli

Bu repo, React + Vite tabanlı bir admin dashboard projesidir.

## Kurulum

```bash
git clone <REPO_URL>
cd qr-system-react
npm install
```

> `<REPO_URL>` kısmına kendi repo adresini yaz.

## Geliştirme

```bash
npm run dev
```

Tarayıcıdan şu adrese git:

```text
http://localhost:5173
```

## Production Build

```bash
npm run build
```

Oluşan `dist/` klasörünü dilediğin sunucuya deploy edebilirsin.
